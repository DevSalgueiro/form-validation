# About the project

Study of a form created with <strong>HTML</strong>, <strong>CSS</strong>, and <strong>JavaScript</strong>.<br><br>The idea is to validate the information filled in by the user. HTML was used to create the form structure, CSS for styling the components, and JavaScript for data validation.
